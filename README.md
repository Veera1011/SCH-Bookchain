# SCH BookChain - Corporate Library Management System

SCH BookChain is a premium, interactive library management application designed for the SCH orchestration network. It offers a seamless experience for employees to discover and borrow resources, while providing administrators with powerful oversight and analytics tools.

## ✨ Key Features
- 🚀 **Modern Discovery**: Responsive masonry grid and featured carousels.
- 📖 **Interactive Details**: 3D page-flip animations for a realistic book-reading feel.
- 🤖 **AI Assistant**: Intelligent library companion powered by Google Gemini.
- 📊 **Admin Dashboard**: Visual analytics, inventory control, and user management.
- 🔒 **Secure Onboarding**: Admin-reviewed registrations with location-based resource locking.

## 🛠 Tech Stack
- **Frontend**: Flutter (Riverpod, GoRouter, Responsive Framework)
- **Backend**: Supabase (PostgreSQL, Auth, Realtime)
- **AI**: Google Generative AI (Gemini API)
- **Design**: Modern Glassmorphism aesthetic with custom SCH branding.

## 🚀 Getting Started

### Prerequisites
- Flutter SDK (Latest Stable)
- Supabase Project
- Gemini API Key

### Configuration
1. Clone the repository.
2. Configure your `.env` file with Supabase and Gemini credentials.
3. Run `flutter pub get` to install dependencies.

### Running the App
```bash
flutter run
```
